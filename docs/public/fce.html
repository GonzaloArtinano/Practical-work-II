Fundamentals of Computer Engineering (FCE)

In this subject, we learned about many important topics in computer engineering, such as:

  • History of Computer Engineering: A look at how computer engineering started and how it has changed over time.
  • Jobs: How computer engineering is used in different jobs and industries today.
  • Binary Numbers: Understanding the binary number system and its use in computers.
  • HTML: Basics of HTML, the language used to create web pages.
  • CSS: Basics of CSS, used to style and design web pages.

These topics give students a good starting point for more advanced studies in computer engineering.
